Проект TaskTracker – это консольное приложение на C# для управления списком задач.Реализованы следующие функции:

    Добавление, просмотр, редактирование, удаление задач.

    Изменение статуса задачи (New / InProgress / Done).

    Поиск по названию, фильтрация по статусу, сортировка списка.

    Валидация вводимых данных (название не пустое, максимальная длина).

Проект разработан в учебных целях для изучения основ C# и принципов командной разработки.

# TaskTracker — Консольный менеджер задач

[![.NET Tests](https://github.com/123wask/TaskTracker2/actions/workflows/dotnet-tests.yml/badge.svg)](https://github.com/123wask/TaskTracker2/actions/workflows/dotnet-tests.yml)

**TaskTracker** – это учебное консольное приложение на C# для управления списком задач. Проект разработан в рамках лабораторных работ по дисциплине «Поддержка и тестирование программных модулей».

## 📌 Возможности

- ✅ Добавление, просмотр, редактирование и удаление задач
- ✅ Изменение статуса задачи: `New` (новая), `InProgress` (в работе), `Done` (готова)
- ✅ Поиск по названию
- ✅ Фильтрация по статусу
- ✅ Сортировка списка (по ID, по статусу)
- ✅ Подсчёт статистики (количество задач по статусам)
- ✅ Валидация ввода (название не пустое, максимальная длина)
- ✅ Логирование действий в консоль
- ✅ Непрерывная интеграция через GitHub Actions (автоматическая сборка и тестирование при каждом push)

## 🛠 Технологии

- C# / .NET 8
- GitHub Actions (CI)
- MSTest (юнит-тестирование)
